Addendum

This project implements a **Mad Lib generator** with a unique time travel theme. The ExpressJS server processes POST form data and returns a custom response. Key highlights:

  • Express Middleware: Used express.urlencoded() for easy form parsing.
  • Custom Route: Handled POST requests at /ITC505/lab-7/submit.
  • Server Logging: Integrated morgan for clean server logs.
  • Error Handling: Returns a friendly HTML message for incomplete submissions.
  • Responsive Design: Styled with a soft futuristic look using CSS gradients and shadows.
